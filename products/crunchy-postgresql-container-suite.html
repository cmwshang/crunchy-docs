PostgreSQL Container Suite

Trusted Open Source PostgreSQL for Container Platforms

Many enterprises are finding containers to be efficient and effective as they scale out and make data services more responsive. Crunchy PostgreSQL Container Suite is a collection of state-of-art certified docker container images and microservices required to deploy 100% Open Source Crunchy Certified PostgreSQL and enterprise PostgreSQL functionality in a container platform architecture including:

  • Backup, Restore and Disaster Recovery
  • High Availability
  • Automated Failover
  • Geospatial Data Management
  • Integrated Administration Dashboard
  • Database Monitoring
  • DBA Utilities

Crunchy PostgreSQL Container Suite brings production PostgreSQL to any enterprise “containerized” environment and it is supported by the same Crunchy 24x7 service level commitment.

More than Just a PostgreSQL Container


Enterprise production PostgreSQL deployments require more than just a PostgreSQL database – enterprises require advanced capabilities like high availability, backup/restore, disaster recovery and monitoring. Similarly, production PostgreSQL deployments on container platforms require more than just a PostgreSQL container. Crunchy PostgreSQL Container Suite provides the microservices necessary to production ready PostgreSQL to meet your performance requirements in a containerized environment.

Recognized and Certified for Leading Container Technologies

Crunchy PostgreSQL Container Suite is cloud agnostic and certified for production deployment with leading container platform technologies. By leveraging leading container and platform technologies, Crunchy PostgreSQL Container Suite supports all major public clouds and enables hybrid in on-premise and public cloud environments.

Backed by Crunchy Secure Enterprise Support, enterprises have the confidence of 24x7 Crunchy PostgreSQL Engineering expertise to support production Crunchy PostgreSQL Container Suite.
